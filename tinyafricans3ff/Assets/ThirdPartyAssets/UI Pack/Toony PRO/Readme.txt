UI Pack Toony PRO 1.1.1
------------------------------------------------------------------

	This package contains 300+ textures, 80+ UI prefabs of cartoon style UIs. There are 26 sample scripts and 20 demo scenes.

	Plus, GUI Animator for Unity UI is included, it lets you easily animate all UI objects like a professional.

	Features:

		• 80+ UI prefabs.
		• 300+ psd files (UHD size).
		• 20 demo scenes for Portrait and Landscape orientations.
		• 26 sample C# scripts.
		• 15 Particle that compatible with Unity UI Canvas.
		• 39 sounds.
		• GUI Animator for Unity UI is included.

		• Support all build player platforms.
		
	Compatible:

		• Unity 4.7.1 or higher.
		• Unity 5.3.4 or higher.

	Product page:
